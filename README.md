# ReelVault

A small web app I made to save and organize Instagram Reels. Built with plain HTML, CSS, and JavaScript — no frameworks or libraries.

# What it does

You paste an Instagram Reel link, pick a category, optionally add a note, and it saves the reel in your browser. It uses the noembed API to grab the title, author name, and thumbnail automatically.

# Main features

- Save reels with category and notes
- Search by title, author, or note
- Filter by category
- Sort by date, author name, or favorites
- Mark reels as favorite
- Dark mode / light mode
- Everything saves in localStorage so it works without any backend

# Files

- `index.html` — the page layout
- `style.css` — styling and theme
- `app.js` — all the logic

# How to run

Just open `index.html` in a browser. That's it.

# Built with

- HTML, CSS, JavaScript
- noembed.com API for fetching reel metadata
- localStorage for saving data
